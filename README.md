# Breast Cancer Gene Expression Analysis

![Project Poster](./final%20poster.png)

## Overview

This project analyzes the GSE45827 breast cancer dataset using Python and bioinformatics approaches to identify differentially expressed genes associated with basal breast cancer.

## Methods

- Differential Gene Expression Analysis
- PCA (Principal Component Analysis)
- Heatmap Visualization
- Hierarchical Clustering (Dendrogram)
- Volcano Plot Analysis
- Functional Enrichment Analysis

## Key Findings

The analysis identified several highly upregulated genes in basal breast cancer, including:

- RRM2
- CXCL10
- TOP2A
- CDC20

These genes are associated with cell proliferation and cancer progression.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset

GEO Dataset: GSE45827
