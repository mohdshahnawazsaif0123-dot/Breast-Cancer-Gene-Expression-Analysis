# Breast Cancer Gene Expression Analysis

## Overview
This project analyzes the GSE45827 breast cancer dataset using Python and bioinformatics approaches.

## Methods
- Differential Gene Expression Analysis
- PCA
- Hierarchical Clustering
- Heatmap
- Volcano Plot
- GO Enrichment Analysis
- Reactome Analysis

## Key Findings
- RRM2
- CXCL10
- TOP2A
- CDC20
- UBE2C

## Significant Pathways
- Cell Cycle
- Cell Division
- Chromosome Segregation
- Mitotic Processes

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- g:Profiler
